## LoRA가 하는 것

쉽게 말하면 **기존 모델을 건드리지 않고 특정 방향으로 살짝 비틀어주는 것**이에요.

```
기본 모델: 모든 것을 그릴 수 있는 일반 AI
LoRA 적용: "이 방향으로 더 치우쳐서 그려"
```

내부적으로는 모델의 가중치에 **작은 행렬을 더해서** 특정 스타일, 개념, 인물 등을 강화해요.

---

## 종류별로 보면

|LoRA 종류|참고하는 것|
|---|---|
|스타일 LoRA|특정 화풍, 색감, 질감|
|캐릭터 LoRA|특정 인물의 외형|
|컨셉 LoRA|특정 개념 (나이, 감정 등)|
|텍스쳐 LoRA|특정 재질감|

---

## ComfyUI에서 비슷한 기능들

| 기능                    | 역할                | LoRA와 차이   |
| --------------------- | ----------------- | ---------- |
| **LoRA**              | 모델 가중치 조정         | 학습된 데이터 기반 |
| **IP-Adapter**        | 참조 이미지의 스타일/내용 적용 | 이미지를 직접 참조 |
| **ControlNet**        | 구도, 포즈, 엣지 제어     | 구조적 제어에 특화 |
| **Textual Inversion** | 새로운 개념을 토큰으로 학습   | LoRA보다 가벼움 |
| **Style Model**       | 스타일만 추출해서 적용      | Flux 전용    |

---

## 한 줄 차이점

```
LoRA       → 모델 자체를 살짝 수정
IP-Adapter → 이미지를 보여주면서 "이렇게 해줘"
ControlNet → 구조/포즈를 강제로 지정
```

가장 비슷한 건 **IP-Adapter**예요. LoRA는 학습 데이터 기반이고, IP-Adapter는 참조 이미지 기반이라는 차이가 있어요 🙌
