---

## 개요
### 이 모델이란?
Meta AI가 개발한 Segment Anything Model의 3세대 버전. 이미지 내 객체를 텍스트 프롬프트 또는 클릭 포인트만으로 정밀하게 분리할 수 있는 범용 세그멘테이션 모델이다.

### 어떤 문제를 해결하는가?
배경 제거, 특정 객체 마스킹, 인페인팅 준비 등 이미지에서 원하는 영역만 분리해야 할 때 사용. 특히 색상이 비슷하거나 경계가 불명확한 경우에도 형태/컨텍스트 기반으로 인식한다.

### 이전 버전 / 유사 모델과의 차이점
| 모델                  | 차이점                        |
| ------------------- | -------------------------- |
| SAM1                | 클릭 포인트만 지원, 텍스트 불가         |
| SAM2                | 비디오 세그멘테이션 지원 추가           |
| **SAM3**            | 텍스트 프롬프트로 객체 지정 가능, 정확도 향상 |
| GroundingDINO + SAM | SAM3 단독으로 대체 가능            |
| RemBG               | SAM3가 복잡한 경계에서 훨씬 정교함      |

---

## 기능 및 역할
### 하는 일
- 텍스트 입력("leaf", "person" 등)으로 객체 자동 감지 및 마스크 생성
- 클릭 포인트 기반 세그멘테이션
- 복잡한 외곽선(머리카락, 잎 톱니 등) 정밀 처리

### 주요 사용처
- 배경 제거 → 투명 PNG 추출
- 인페인팅을 위한 마스크 생성
- 특정 객체만 추출해서 다른 워크플로우에 연결
- 나뭇잎, 인물 등 자연물 세그멘테이션

### 못하는 것 / 한계
- 색상/형태가 매우 유사한 객체와 배경 분리 어려움
- 부분적으로 가려진 객체는 누락될 수 있음 (예: 헤드셋)
- 누락 부분은 멀티 프롬프트 또는 마스크 합치기로 보완 필요

## 설치
### 필요한 것
- ComfyUI 최신 버전
- VRAM 최소 요구: 4GB 이상 권장
- 의존성: 없음 (Manager로 설치 시 자동)

### 설치 방법
1. Manager → Custom Nodes → `SAM3` 검색 → Install
2. 모델 파일 자동 다운로드 (최초 실행 시)
3. 모델 파일 경로: `ComfyUI/models/sam/`
4. 재시작 필요: ✅

---

## 사용 방법
### 기본 워크플로우
```
Load Image
→ SAM3 Segmentation (텍스트 입력: "leaf")
→ MaskPreview (확인용)
→ MaskBlur (경계 부드럽게)
→ Apply Mask
→ Save Image (PNG, alpha 포함)
```

### 주요 파라미터
| 파라미터 | 설명 | 권장값 |
|---------|------|--------|
| `value` | 감지할 객체 텍스트 | "leaf", "person" 등 |
| `threshold` | 감지 민감도 | 0.3~0.5 |
| `sam_model` | 사용할 SAM 모델 크기 | base / large |
| `chunk_size` | 처리 단위 | 기본값 유지 |

### 입력 / 출력
| | 타입 | 설명 |
|--|------|------|
| **입력** | IMAGE | 원본 이미지 |
| **입력** | TEXT | 감지할 객체 텍스트 |
| **출력** | IMAGE | 세그멘테이션 결과 이미지 |
| **출력** | MASK | 생성된 마스크 |

## 마스크 후처리 조합
| 노드 | 포토샵 대응 | 용도 |
|------|-----------|------|
| `MaskBlur` | 페더 | 경계 부드럽게 |
| `GrowMask` | 확장 | 마스크 영역 확장 |
| `MaskErode` | 수축 | 경계 안쪽으로 줄이기 |
| `MaskComposite` | 레이어 합치기 | 여러 마스크 합치기 |

---

## 연관 모델 / 노드
| 이름 | 관계 | 용도 |
|------|------|------|
| GroundingDINO | 대체/조합 가능 | 텍스트 기반 객체 감지 |
| RemBG | 단순 대안 | 빠른 배경 제거 |
| BiRefNet | 보완 | 정밀 외곽선 처리 |
| MaskBlur | 후처리 | 경계 페더링 |
| CHORD | 연결 활용 | 배경 제거 후 PBR 맵 추출 |

## 주의사항 / 알려진 문제
- 헤드셋처럼 사람 몸에 붙은 물체는 별도 프롬프트 필요
- 자연물(나뭇잎)은 비슷한 색상 배경에서 누락 가능
- 누락 시: 멀티 프롬프트 또는 MaskComposite로 보완
