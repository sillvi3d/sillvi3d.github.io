## 크레딧 활용 전략

### 무료 생성량 추산

|서비스|무료 크레딧|월 가능 생성 수|
|---|---|---|
|Hunyuan3D|매일 20회 리셋|최대 ~600회 ★|
|Tripo3D|월 300크레딧|10~30회 (모드별 상이, 직접 확인)|
|Meshy|월 200크레딧 추정|미확인 (직접 확인)|

### 일일 루틴

**Hunyuan3D — 매일 소진 우선**

- 매일 20회 → 같은 오브젝트 3~4회 돌려 베스트 픽 선정
- 자연물 / 식물 / 모듈형 위주
- 아웃풋 GLB → 언리얼 바로 임포트 테스트

**Tripo3D — 퀄리티 필요할 때만**

- 크레딧 소모량 직접 확인 후 일일 사용량 역산
- Hunyuan 결과가 부족한 복잡한 오브젝트에 사용
- FBX 아웃풋 → LOD 설정 용이

## 서비스별 특징

### Hunyuan3D (Tencent)

- 웹 UI : https://3d.hunyuan.tencent.com
- 매일 20회 무료 — 현재 가장 관대한 무료 플랜
- 텍스트 / 단일 이미지 / 멀티뷰 인풋 모두 지원
- 로컬 실행 가능하나 12GB+ VRAM 권장 (8GB 어려움)

### Tripo3D

- 웹 UI : https://www.tripo3d.ai
- 월 300크레딧, 생성 모드별 소모량 상이 → 직접 테스트 필요
- FBX 지원으로 언리얼 파이프라인에 유리
- 리메시, 자동 리깅 등 후처리 기능 내장

### Meshy

- 웹 UI : https://www.meshy.ai
- 텍스처 퀄리티가 타 서비스 대비 우수하다는 평가
- Text to Texture 기능(기존 메시에 텍스처만 생성) 강점

### Rodin / Hyper3D

- 웹 UI : https://hyper3d.ai
- 고퀄리티 출력 특화, 무료 크레딧 적음
- 게임 애셋용 커뮤니티 평가 높음

### CSM (Common Sense Machines)

- 웹 UI : https://3d.csm.ai
- 멀티뷰 인풋 특화 — 삼면도 인풋 시 정확도 높음
- 포인트 클라우드 방식으로 디테일 보존 강점

### Luma AI Genie

- 웹 UI : https://lumalabs.ai/genie
- 빠른 생성 속도
- USDZ 포맷 지원 (AR / 언리얼 호환)
- 타 서비스 대비 디테일 낮은 편
