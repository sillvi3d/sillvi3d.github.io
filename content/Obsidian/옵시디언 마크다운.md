##### 제목

```
## 제목 2
### 제목 3
#### 제목 4
```

---

##### 텍스트 스타일

```
**굵게**
*기울임*
~~취소선~~
==하이라이트==
**_굵게 + 기울임_**
```

---

##### 목록

```
- 순서 없는 목록
  - 들여쓰기

1. 순서 있는 목록
2. 두 번째
```

---

##### 체크박스

```
- [ ] 할 일
- [x] 완료된 일
```

---

##### 링크

```
[텍스트](URL)                  외부 링크
[[노트이름]]                   내부 링크
[[노트이름|표시텍스트]]         별칭 링크
[[노트이름#헤딩]]               헤딩으로 이동
```

---

##### 이미지

```
![[이미지파일.png]]            내부 이미지
![[이미지파일.png|300]]        너비 지정
![설명](URL)                  외부 이미지
```

##### 표

```

|-------|-------|-------|
| 내용1 | 내용2 | 내용3 |
```

---

##### 각주

```
본문[^1]

[^1]: 각주 내용
```

---

##### 수식 (LaTeX)

```
인라인: $수식$
블록: $$수식$$
```

---

##### 옵시디언 전용

```
[[노트이름]]                   내부 링크 (백링크 생성)
![[노트이름]]                  노트 임베드
#태그                          태그
^블록ID                        블록 참조
[[노트이름#^블록ID]]            블록 링크
```

---

##### 콜아웃 (Callout)

```
> [!note] 제목
> 내용

> [!tip] 팁
> [!warning] 경고
> [!danger] 위험
> [!info] 정보
> [!success] 성공
> [!question] 질문
> [!quote] 인용
```
